
# 📈 A股账户对账单自动分发系统 (Auto Email Sender RPA)

这是一个基于 Python 开发的轻量级办公自动化 (RPA) 脚本。专门用于券商/私募等金融场景下，每日按时、精准地将 A 股账户对账单自动通过邮件分发给对应的客户。

## ✨ 核心特性

* **🗓️ 智能交易日推算**：内置中国法定节假日与工作日历，自动过滤周末与休市日，精准定位 **T-1** 交易日，自动抓取对应日期的文件夹。
* **⚙️ 代码与配置彻底分离**：
  * **系统配置 (`config.ini`)**：支持自定义邮件服务器、多时间点调度表（如 09:15, 12:00 等）以及失败重试策略。
  * **名单热更新 (`email_mapping.csv`)**：业务人员可直接用 Excel 维护“资金账号-收件邮箱”映射表，程序每次运行时动态读取，无需重启代码。
* **📦 智能打包与多附件支持**：自动根据“资金账号”提取前缀，将同一客户的多个附件（Excel、RAR、ZIP 等）智能打包进同一封邮件中。
* **🛡️ 容错与防重机制**：
  * 发送成功的账号会被记录在案，同一批次绝不重复发送。
  * 遇到网络或邮箱服务器异常时，支持**批处理统一重试**机制（如等待 5 分钟后重试失败名单）。
  * 完善的按日切分日志系统（正常日志与 Error 日志独立记录）。

## 🚀 快速开始

### 1. 环境依赖
确保您的系统已安装 Python 3.x，然后在终端中运行以下命令安装所需依赖：
\`\`\`bash
pip install schedule chinesecalendar
\`\`\`

### 2. 配置准备
1. 复制一份 \`config.ini\` 并填入您的真实 SMTP 邮箱账号和授权码。
2. 准备一份 \`email_mapping.csv\`，包含 \`资金账号\` 和 \`收件人邮箱\` 两列内容。

### 3. 运行程序
\`\`\`bash
python auto_email_sender.py
\`\`\`
程序启动后将进入守护模式，在您设定的 `config.ini` 时间点自动唤醒执行。
